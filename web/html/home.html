Medication & Pharmacological Resource Portal 0.1

This is a resource service intended to help other developers focus on their own application and not on data extraction and manipulation. The main features of the service are listed below.


Remote Procedure Calls

To make full use of the underlying Prolog entailment engine you can do remote procedure calls (RPC) to this server. Through this method you can treat this service as a local source to your Prolog or Javascript program. This enables you to do arbitrarily complex queries.

API

The service is available for queries through the API. These queries return data meant for machine consumption.

Explore data store

You can browse the graphs and use the interactive search form to learn about the ontologies and available resources in the data store.

SPARQL

For testing and learning about SPARQL queries you can try one of the three SPARQL endpoints accessible from the menu. Pick your favourite. The "Simple Form" endpoint also handles SeRQL queries.
